Calculadora de Velocidad Crítica

Introduce entre 3 y 5 marcas del corredor usando distancias bien medidas y tiempos reales. A partir de esos datos, la herramienta calcula la velocidad crítica y genera estimaciones de ritmo por distancia y por tiempo.

  • Usa esfuerzos máximos o casi máximos, no rodajes suaves ni entrenamientos incompletos.
  • Cuanto más consistentes sean los datos, más útil será el resultado.
  • Es recomendable usar marcas de diferentes duraciones para mejorar el ajuste.
  • La herramienta ofrece una estimación, no una predicción exacta para cualquier distancia.

Nota

Esta herramienta estima el rendimiento a partir del modelo de velocidad crítica. Funciona mejor con esfuerzos bien medidos y realizados con alta intensidad. Si el ajuste sale extraño o el valor de D′ es negativo, conviene revisar si las marcas introducidas son coherentes entre sí.
