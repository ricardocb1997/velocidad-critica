Nota

Esta herramienta estima el rendimiento a partir del modelo de velocidad crítica. Funciona mejor con esfuerzos máximos o casi máximos, bien medidos, y dentro de un rango razonable. No debe interpretarse como una predicción perfecta para todas las distancias.
